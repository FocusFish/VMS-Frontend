User manual

Search

The search field is used for centering the map on a specific ship.
You can search ships by name in this field and once you have chosen the desired ship in the dropdown the map will center around that ship. The dropdown is sorted alphabetically and shows the top 10 results.
The search will try to find any ship containing that letter combination. For Example the search query as will give you ship sush as Askungen but also ships like Basto.

Special commands:

!

By starting with an exlamation mark the search will instead try to find any ship not containing the given text. I.e !as will give the reverse result.

/c

By starting your command with /c you can go directly to a specific cordinate. Example: /c 58.43562 17.45620

Filter

The filter field is used for filtering ships that will be shown on the map. Just like the search function it will try to find any ship with the text in the name per default.

Special commands:

!

By starting with an exlamation mark the filter will instead try to find any ship not containing the given text.

/f

By starting with /f the filter will instead filter on flagstate.
Examples:
  • /f swe will give you all swedish ships.
  • /f swe,dnk will give you all swedish and danish ships.
  • /f !swe,dnk will give you all ships that are not swedish or danish.

&

Is a separator, by using it you can filter on multiple things. All arguments given to the filter must be true for the ships to be visible.
Examples:
  • /f swe & ask will give you all swedish ships that have ask in the name.
  • ask & /f swe,dnk will give you all swedish and danish ships that have ask in the name.
  • !ask & /f swe,dnk will give you all swedish and danish ships that dosn't have ask in the name.
